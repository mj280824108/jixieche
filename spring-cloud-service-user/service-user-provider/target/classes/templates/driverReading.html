司机是由车主聘用的，帮助其泥头车跑趟的人。
司机被绑定到某辆车上后，司机可以为该车辆接(兼职司机除外)，并去执行订单。
在平台司机有三种类型：
一、车主本人（我自己）；
二、通过平台发布兼职职位招聘到的兼职司机（兼职司机）；
三、车主通过平台招聘的全职司机或通过自有渠道招聘的司机(我的司机）。
第一类：车主本人（我自己）是注册后自动生成，不可删除的；
第二类：（兼职司机）是通过平台发布兼职职位招聘到的司机，达到职位输入的工作时间后，可选择兼职司机绑定到某辆车上，该司机则可去执行订单；
第三类：（我的司机）是车主通过平台招聘的全职司机或通过自有渠道招聘的司机，招聘到该司机后需要通过添加司机页面添加司机才可绑定司机给自己的某辆泥头车接单、执行订单。
